E-Commerce Customer Behavior Analytics

End-to-End Data Analytics Project using Python, MySQL, SQL, and Power BI

Overview

This project focuses on analyzing customer shopping behavior using an e-commerce transaction dataset. The goal is to convert raw customer data into meaningful business insights that can support better decision-making.

The project follows a complete data analytics workflow, starting with data preprocessing in **Python**, storing the cleaned data in **MySQL**, performing business analysis using **SQL**, and finally building an interactive dashboard in **Power BI**. The analysis highlights customer purchasing patterns, product performance, revenue trends, subscription behavior, and the impact of discounts on sales.

Project Steps

1. Data Loading

The dataset was imported into Python using the **Pandas** library. Initial checks were performed to understand the dataset structure, data types, and summary statistics.

2. Exploratory Data Analysis (EDA)

An initial exploration of the dataset was carried out to understand customer demographics, purchase behavior, and product information. Missing values, data distributions, and key statistics were examined before preprocessing.

3. Data Cleaning

To improve data quality, missing values were handled, column names were standardized, duplicate records were checked, and the dataset was validated for consistency before analysis.

4. Feature Engineering

Two additional features were created to enhance the analysis:

- **Age Group** – Customer age segmentation
- **Purchase Frequency (Days)** – Numerical representation of purchase frequency

5. Database Integration

The cleaned dataset was uploaded to **MySQL Server** using **SQLAlchemy**, allowing SQL queries to be executed for business analysis.

6. SQL Business Analysis

Several SQL queries were written to answer important business questions, including:

- Revenue by Gender
- High-Spending Customers Using Discounts
- Top Rated Products
- Shipping Method Comparison
- Subscribers vs Non-Subscribers
- Customer Segmentation
- Revenue by Age Group
- Customer Lifetime Value (CLV)
- Revenue by Location
- Discount Impact Analysis

7.Dashboard

An interactive **Customer Behavior Dashboard** was developed in **Power BI** to provide a comprehensive overview of customer purchasing behavior, revenue distribution, and demographic trends. The dashboard enables users to monitor key business metrics and explore customer insights using interactive filters.

Key Performance Indicators (KPIs)

The dashboard includes the following KPIs:

- **Number of Customers:** 3.9K
- **Average Purchase Amount:** $59.76
- **Average Review Rating:** 3.75

Dashboard Visualizations

The dashboard includes the following visualizations:

- **Customer Distribution by Subscription Status**
  - Displays the percentage of subscribed and non-subscribed customers.
  - Helps understand customer membership trends.

- **Revenue by Product Category**
  - Compares total revenue generated by each product category.
  - Identifies the highest revenue-generating categories.

- **Sales by Product Category**
  - Shows the number of purchases made in each category.
  - Highlights the most popular product categories.

- **Revenue by Age Group**
  - Visualizes revenue contribution across different customer age groups.
  - Helps identify high-value customer segments.

- **Sales by Age Group**
  - Compares the number of purchases made by each age group.
  - Provides insights into customer purchasing patterns.

Interactive Filters

The dashboard allows users to dynamically filter the analysis using:

- Subscription Status
- Gender
- Product Category
- Shipping Type

These slicers enable users to interactively explore customer behavior across different customer segments and business dimensions.


Results

The analysis uncovered several meaningful business insights:

- Identified customer groups contributing the highest revenue.
- Compared purchasing behavior between subscribed and non-subscribed customers.
- Determined the highest-rated and most frequently purchased products.
- Evaluated the effect of discounts on customer spending.
- Identified locations contributing the highest revenue.
- Segmented customers based on their purchase history.
- Analyzed spending patterns across different age groups.

These insights can support better pricing strategies, targeted marketing campaigns, customer retention initiatives, and overall business growth.
